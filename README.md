# gdrive_cli

## Install 
`pip install -r requirements.txt`

## Run
```
(.venv)> python run.py -h
INFO: Showing help with the command 'run.py -- --help'.

NAME
    run.py

SYNOPSIS
    run.py COMMAND

COMMANDS
    COMMAND is one of the following:

     trash_delete
       not implemented

     trash_restore
       restores all items currently in the trash
```
